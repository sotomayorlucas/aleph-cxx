## Areas audited — clean

- **Lowering incremental multi-op chains**: byte-identical to full lower (512× stress).
- **DPO apply + invariants**: random graph stress, all pass `validate_all`.
- **Thread pool**: no counter corruption under parallel_for stress.
- **Sparse LDLᵀ**: negative pivot returns `NotPsd`; near-singular accepted as kernel direction.

## Recommendations

1. **Permanent fuzz**: libFuzzer harness on `load_obj` + DPO `apply` with `validate_all` oracle.
2. **Window API**: consider `expected<Window, string>` factory instead of `abort()` for embedders.
3. **Sheaf asserts**: remaining `assert()` in cohomology/cochain paths should migrate to `expected` (deferred — large surface).
4. **CI**: add headless ctest job; skip SDL tests when `SDL_Init` fails.
